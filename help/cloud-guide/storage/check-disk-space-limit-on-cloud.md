# Vérifier la limite d’espace disque pour Adobe Commerce sur l’infrastructure cloud

Pour vérifier l’espace disponible pour chaque Adobe Commerce dans l’environnement d’infrastructure cloud :

1. Connectez-vous à [Cloud Console](https://console.adobecommerce.com/).
2. Sélectionnez le projet approprié dans le tableau de bord **Tous les projets**. Dans le coin inférieur, consultez Disponibilité de l’espace disque.
